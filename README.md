# DMC Statistical Resources: The Madison School

This project is a curated resource hub for Data Monitoring Committee (DMC) statisticians, focusing on the independence models pioneered by Dave DeMets and the UW-Madison SDAC.

## 🌐 View the Website
**For the cleanest reading experience, visit the live site here:**
[https://bmbadger.github.io/dmc-resources/](https://bmbadger.github.io/dmc-resources/)

## Contents
- **DeMets Classics:** A bibliography of foundational papers (Fisher, Lan, etc.).
- **Reporting Model:** A deep dive into the Open vs. Closed report firewall.
